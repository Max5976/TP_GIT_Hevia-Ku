# TP – Desarrollo Colaborativo con Git y GitHub

## Proyecto: Ideas para el Último Año

Este repositorio contiene el **proyecto base** para el trabajo práctico de Git y GitHub.

La página funciona… pero seamos sinceros: **está bastante fea y tiene varios errores**.
Hay cosas mal escritas, estilos mejorables y código que necesita cariño.

La misión de cada grupo será **arreglar el proyecto, mejorarlo y trabajar colaborativamente usando Git**.

---

## Archivos del proyecto

El repositorio contiene una página web simple con:

* `index.html`
* `style.css`
* `script.js`

---

## Objetivo del trabajo

Durante el trabajo práctico deberán:

* Corregir errores del proyecto
* Mejorar el diseño de la página
* Mejorar la funcionalidad con JavaScript
* Trabajar colaborativamente usando **Git y GitHub**

---

## Cosas que se pueden mejorar

Algunas ideas para trabajar en el proyecto:

### HTML

* Corregir etiquetas
* Mejorar la estructura del documento
* Agregar secciones nuevas

### CSS

* Corregir errores de sintaxis
* Mejorar el diseño
* Agregar efectos como `hover`
* Hacer el diseño responsive

### JavaScript

* Corregir errores en condiciones
* Validar formularios
* Mostrar mensajes al usuario
* Mejorar la interacción de la página

---

##  Trabajo en equipo

Cada integrante del grupo trabajará en **su propia rama** y luego integrarán los cambios utilizando **Pull Requests y Merge**.

La idea es simular cómo trabajan los equipos reales de desarrollo.

---

##  Misión

Transformar esta página simple en algo **mucho más prolijo, funcional y copado** usando Git.

Arreglen errores, mejoren el diseño, agreguen ideas y dejen el proyecto mejor de lo que lo encontraron.
